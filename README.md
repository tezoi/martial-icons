# martial-icons
Material icons for martial arts and other sports

![icons](all.png)

The SVGs were created using Inkscape.
They include a green belt.

Actually available icons are :

 - horse stance,
 - cat stance,
 - sited meditation,
 - dragon stance,
 - high kick.
